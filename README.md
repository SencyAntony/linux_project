# linux_project

Linux Commands

1. mkdir EV
Creates a new directory named EV.
2. cd EV
Changes the current working directory to EV.

3. mkdir 52
Creates a directory with the given roll number inside EV.

4. cd 52
Moves into that directory.

5. cd /
Navigates to the root directory of the Linux file system.

6. pwd
Displays the absolute path of the current working directory.

7. ls -l
Lists files and directories.

8. ls -al
Lists all files including hidden files

9. rm -i file1
Deletes file1 after asking for user confirmation.

10. rmdir file
Removes the directory file since it is empty.

11. rm -r file2
Deletes the directory file2 along with all its contents.

12. cat > file1.txt
Creates file1.txt and allows the user to enter text from the terminal.

13. cat > file2.txt
Creates file2.txt and stores user-entered text.

14. mv file_c.txt combined
Renames the file to combined.

15. chmod u+x combined
Gives execute permission to the file owner.
